# 第三大題 / Problem 3

## 📖 Original Text / 原文

**3. Ordering by asymptotic growth rates:**

&nbsp;&nbsp;$n! \quad (\lg n)! \quad \lg(n!) \quad \lg^* n \quad 2^{\lg n} \quad 2^n \quad n^2$

&nbsp;&nbsp;Hint: Stirling's approximation

## 🇹🇼 Chinese Translation / 中文翻譯

**3. 按漸近成長速率排序：**

&nbsp;&nbsp;$n! \quad (\lg n)! \quad \lg(n!) \quad \lg^* n \quad 2^{\lg n} \quad 2^n \quad n^2$

&nbsp;&nbsp;提示：使用 Stirling 近似公式

## 💡 Detailed Explanation / 詳細解釋

**需要排序的七個函數：**

| 函數 | 名稱 | 直覺成長速率 |
|------|------|-------------|
| $\lg^* n$ | 迭代對數 | 極慢（對任何實際輸入 ≤ 5） |
| $2^{\lg n}$ | 等於 $n$ | 線性 |
| $\lg(n!)$ | 階乘的對數 | 約 $n \lg n$（Stirling） |
| $n^2$ | 多項式 | 二次 |
| $2^n$ | 指數 | 指數成長 |
| $(\lg n)!$ | 對數的階乘 | 超指數（比 $2^n$ 慢） |
| $n!$ | 階乘 | 極快（比任何指數都快） |

**Stirling 近似公式：** $n! \approx \sqrt{2\pi n} \left(\frac{n}{e}\right)^n$，取對數後 $\lg(n!) \approx n \lg n - n \lg e$

---

## 🔢 Derivation Process / 推導過程

### 步驟 1：分析每個函數的成長速率

| 函數 | 簡化 | 依據 |
|------|------|------|
| $\lg^* n$ | $\approx O(1)$ | 迭代對數：$\lg^* n = \min\{i : \lg^{(i)} n \leq 1\}$ |
| $2^{\lg n}$ | $= n$ | $2^{\lg n} = n^{\lg 2} = n^1 = n$ |
| $n^2$ | $n^2$ | 多項式 |
| $\lg(n!)$ | $\Theta(n \lg n)$ | Stirling：$\lg(n!) \approx n \lg n - n \lg e$ |
| $2^n$ | $2^n$ | 指數 |
| $(\lg n)!$ | 超指數 | 見下方推導 |
| $n!$ | $\approx (n/e)^n$ | Stirling：$n! \approx \sqrt{2\pi n}(n/e)^n$ |

### 步驟 2：比較關鍵對

**$\lg^* n$ vs $n$：**

$\lg^*(2^{65536}) = 5$，而 $2^{65536}$ 是天文數字。$\lg^* n$ 成長極慢。

**$n$ vs $n^2$：** 顯然 $n < n^2$。

**$n^2$ vs $\lg(n!)$：**

由 Stirling，$\lg(n!) \approx n \lg n$。比較：

$$\lim_{n \to \infty} \frac{n \lg n}{n^2} = \lim_{n \to \infty} \frac{\lg n}{n} = 0$$

所以 $n \lg n = o(n^2)$，即 $\lg(n!) < n^2$ 對大 $n$。

**$(\lg n)!$ vs $2^n$：**

用 Stirling 近似 $(\lg n)!$：

$$(\lg n)! \approx \sqrt{2\pi \lg n} \left(\frac{\lg n}{e}\right)^{\lg n}$$

取對數：

$$\lg((\lg n)!) \approx \lg n \cdot \lg(\lg n) - \lg n \cdot \lg e$$

而 $\lg(2^n) = n$。

比較 $\lg n \cdot \lg(\lg n)$ vs $n$：

$$\lim_{n \to \infty} \frac{\lg n \cdot \lg(\lg n)}{n} = 0$$

所以 $(\lg n)! \ll 2^n$。

**$(\lg n)!$ vs $\lg(n!)$：**

對 $n = 2^{16} = 65536$：

- $\lg(n!) \approx 16 \times 65536 \approx 10^6$
- $(\lg n)! = 16! \approx 2 \times 10^{13}$

所以 $(\lg n)! > \lg(n!)$ 對大 $n$。

### 步驟 3：最終排序（由小到大）

$$\boxed{\lg^* n \ll 2^{\lg n}(=n) \ll \lg(n!)(\approx n \lg n) \ll n^2 \ll (\lg n)! \ll 2^n \ll n!}$$

### 驗證：對 $n = 2^{16} = 65536$

| 函數 | 值 |
|------|-----|
| $\lg^* n$ | 5 |
| $2^{\lg n} = n$ | 65,536 |
| $\lg(n!) \approx n \lg n$ | $\approx 1,048,576$ |
| $n^2$ | $4,294,967,296$ |
| $(\lg n)! = 16!$ | $\approx 2.09 \times 10^{13}$ |
| $2^n = 2^{65536}$ | $\approx 10^{19728}$ |
| $n! = 65536!$ | $\approx 10^{265564}$ |

排序確認正確。∎
