## 📌 核心公式速查表

| 概念 | 公式 |
|------|------|
| 大 Θ 定義 | $c_1 g(n) \leq f(n) \leq c_2 g(n)$ |
| 大 O 定義 | $f(n) \leq c \cdot g(n)$ |
| Stirling 近似 | $n! \approx \sqrt{2\pi n}(n/e)^n$ |
| $\lg(n!)$ | $\Theta(n \lg n)$ |
| $2^{\lg n}$ | $= n$ |
| 等比數列求和 | $\sum_{j=0}^{k-1} r^j = \frac{1-r^k}{1-r}$ |
| $\lg^k n$ vs $n^\epsilon$ | $\lg^k n = o(n^\epsilon)$ |

---

## 📝 重點整理

1. **大 Θ 證明**：需要同時證明上界和下界，找出合適的常數 $c_1, c_2, n_0$
2. **大 O 證明**：只需證明上界，找出 $c, n_0$
3. **對數 vs 多項式**：$\lg^k n = o(n^\epsilon)$ 對任意 $k, \epsilon > 0$
4. **Stirling 近似**：$n! \approx \sqrt{2\pi n}(n/e)^n$，$\lg(n!) = \Theta(n \lg n)$
5. **遞迴展開法**：反覆代入 → 觀察模式 → 等比/等差求和 → 代入終止條件
6. **主定理**：$T(n) = aT(n/b) + f(n)$，此題 $a=1, b=2, f(n)=\Theta(n)$，答案為 $\Theta(n)$
