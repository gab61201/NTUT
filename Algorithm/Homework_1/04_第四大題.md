# 第四大題 / Problem 4

## 📖 Original Text / 原文

**4. Using expanding method to solve the recurrence:**

&nbsp;&nbsp;$T(n) = T(n/2) + \Theta(n)$ with $T(1) = c$, if $n = 1$

## 🇹🇼 Chinese Translation / 中文翻譯

**4. 使用展開法求解遞迴式：**

&nbsp;&nbsp;$T(n) = T(n/2) + \Theta(n)$，初始條件 $T(1) = c$

## 💡 Detailed Explanation / 詳細解釋

**遞迴式：** $T(n) = T(n/2) + cn$（將 $\Theta(n)$ 寫為 $cn$）

**展開法的核心思想：** 反覆代入遞迴式，展開多層，觀察模式，然後求和。

這實際上就是「遞迴樹」方法的代數版本——每一層代表遞迴的一層深度，每層的工作量是 $cn/2^i$。

**直觀理解：** 每次遞迴把問題規模減半，但每一層的工作量也減半（$cn \to cn/2 \to cn/4 \to \dots$），所以總工作量是等比數列求和，收斂到 $O(n)$。

---

## 🔢 Derivation Process / 推導過程

將 $\Theta(n)$ 寫為 $cn$（$c > 0$ 為常數）：

$$T(n) = T(n/2) + cn$$

### 第 1 次展開：

$$T(n) = cn + T(n/2) = cn + c\left(\frac{n}{2}\right) + T\left(\frac{n}{4}\right) = cn + \frac{cn}{2} + T\left(\frac{n}{4}\right)$$

### 第 2 次展開：

$$T(n) = cn + \frac{cn}{2} + \frac{cn}{4} + T\left(\frac{n}{8}\right)$$

### 第 $i$ 次展開：

$$T(n) = cn \sum_{j=0}^{i-1} \left(\frac{1}{2}\right)^j + T\left(\frac{n}{2^i}\right)$$

### 終止條件：

當 $\frac{n}{2^i} = 1$，即 $i = \lg n$ 時：

$$T(n) = cn \sum_{j=0}^{\lg n - 1} \left(\frac{1}{2}\right)^j + T(1)$$

### 等比數列求和：

$$\sum_{j=0}^{\lg n - 1} \left(\frac{1}{2}\right)^j = \frac{1 - (1/2)^{\lg n}}{1 - 1/2} = \frac{1 - 1/n}{1/2} = 2\left(1 - \frac{1}{n}\right)$$

### 代入：

$$T(n) = cn \cdot 2\left(1 - \frac{1}{n}\right) + c = 2cn - 2c + c = 2cn - c$$

### 結論：

$$\boxed{T(n) = \Theta(n)}$$

（更精確地，$T(n) = 2cn - c = \Theta(n)$）∎
