# 第一大題 / Problem 1

## 📖 Original Text / 原文

**1. Prove by finding constants that satisfy the definition of order of magnitude, that**

&nbsp;&nbsp;**a)** $f = \Theta(g)$ if $f(n) = 3n^2 - 7n$ and $g(n) = n^2$

&nbsp;&nbsp;**b)** $2n^2 - 7n = O(n^2)$

## 🇹🇼 Chinese Translation / 中文翻譯

**1. 透過找出滿足階數定義的常數來證明：**

&nbsp;&nbsp;**a)** 若 $f(n) = 3n^2 - 7n$ 且 $g(n) = n^2$，則 $f = \Theta(g)$

&nbsp;&nbsp;**b)** $2n^2 - 7n = O(n^2)$

## 💡 Detailed Explanation / 詳細解釋

**核心概念：** 這道題目要求使用「形式化定義」來證明漸近符號的關係，而非直觀判斷。

**大 Θ（Theta）的定義：**

$f(n) = \Theta(g(n))$ 表示存在正常數 $c_1, c_2, n_0$，使得對所有 $n \geq n_0$：

$$c_1 \cdot g(n) \leq f(n) \leq c_2 \cdot g(n)$$

這表示 $f(n)$ 被 $g(n)$「夾在中間」——既不會比 $g(n)$ 慢太多，也不會比 $g(n)$ 快太多。

**大 O（Oh）的定義：**

$f(n) = O(g(n))$ 表示存在正常數 $c, n_0$，使得對所有 $n \geq n_0$：

$$0 \leq f(n) \leq c \cdot g(n)$$

這表示 $f(n)$ 的成長速率「不超過」$g(n)$ 的成長速率（上界）。

**關鍵直覺：**

- $3n^2 - 7n$ 中，當 $n$ 很大時，$-7n$ 的影響遠小於 $3n^2$，所以主導項是 $3n^2$
- 因此 $3n^2 - 7n$ 的成長速率與 $n^2$ 相同（只是常數倍不同）

---

## 🔢 Derivation Process / 推導過程

### 1a：證明 $f(n) = 3n^2 - 7n = \Theta(n^2)$

**目標：** 找出 $c_1, c_2, n_0 > 0$ 使得對所有 $n \geq n_0$：

$$c_1 \cdot n^2 \leq 3n^2 - 7n \leq c_2 \cdot n^2$$

**上界（右側不等式）：**

$$3n^2 - 7n \leq 3n^2 \quad \text{（因為 } -7n \leq 0 \text{ 對 } n \geq 0\text{）}$$

所以取 $c_2 = 3$，對所有 $n \geq 1$ 成立。

**下界（左側不等式）：**

需要 $c_1 \cdot n^2 \leq 3n^2 - 7n$，即 $c_1 \leq 3 - \frac{7}{n}$

當 $n \geq 7$ 時，$\frac{7}{n} \leq 1$，所以 $3 - \frac{7}{n} \geq 2$

取 $c_1 = 2, n_0 = 7$：

$$2n^2 \leq 3n^2 - 7n \iff 7n \leq n^2 \iff n \geq 7 \quad \checkmark$$

**結論：** 取 $c_1 = 2, c_2 = 3, n_0 = 7$，對所有 $n \geq 7$：

$$2n^2 \leq 3n^2 - 7n \leq 3n^2$$

因此 $f(n) = \Theta(n^2) = \Theta(g(n))$。∎

---

### 1b：證明 $2n^2 - 7n = O(n^2)$

**目標：** 找出 $c, n_0 > 0$ 使得對所有 $n \geq n_0$：

$$0 \leq 2n^2 - 7n \leq c \cdot n^2$$

**非負性：** $2n^2 - 7n \geq 0 \iff 2n \geq 7 \iff n \geq 3.5$，取 $n_0 = 4$。

**上界：**

$$2n^2 - 7n \leq 2n^2 \quad \text{（因為 } -7n \leq 0\text{）}$$

取 $c = 2, n_0 = 4$。

**結論：** 取 $c = 2, n_0 = 4$，對所有 $n \geq 4$：

$$0 \leq 2n^2 - 7n \leq 2n^2$$

因此 $2n^2 - 7n = O(n^2)$。∎
