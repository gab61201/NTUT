# 第二大題 / Problem 2

## 📖 Original Text / 原文

**2. State whether each of the following is true or false:**

&nbsp;&nbsp;**a)** $7n^2 \lg n + 16n = O(n^3)$

&nbsp;&nbsp;**b)** $(\lg n)^k = O(n^4)$, $k$ is a constant

## 🇹🇼 Chinese Translation / 中文翻譯

**2. 判斷以下各敘述為真或假：**

&nbsp;&nbsp;**a)** $7n^2 \lg n + 16n = O(n^3)$

&nbsp;&nbsp;**b)** $(\lg n)^k = O(n^4)$，$k$ 為常數

## 💡 Detailed Explanation / 詳細解釋

**問題 2a：** $7n^2 \lg n$ 的成長速率介於 $n^2$ 和 $n^3$ 之間。因為 $\lg n$ 成長很慢，所以 $n^2 \lg n$ 遠小於 $n^3$。因此這個上界成立。

**問題 2b：** 對數的任何常數次方都遠小於任何正次數的多項式。這是演算法分析中的一個基本事實：$\lg^k n$ 永遠比 $n^\epsilon$ 慢（對任意 $k > 0, \epsilon > 0$）。

---

## 🔢 Derivation Process / 推導過程

### 2a：$7n^2 \lg n + 16n = O(n^3)$？

**答案：True（真）**

**證明：** 需要找 $c, n_0$ 使得 $7n^2 \lg n + 16n \leq c \cdot n^3$

對 $n \geq 2$：

- $\lg n \leq n$（對數永遠小於線性）
- $7n^2 \lg n \leq 7n^2 \cdot n = 7n^3$
- $16n \leq 16n^3$（對 $n \geq 1$）

所以：

$$7n^2 \lg n + 16n \leq 7n^3 + 16n^3 = 23n^3$$

取 $c = 23, n_0 = 2$，即得 $7n^2 \lg n + 16n = O(n^3)$。∎

---

### 2b：$(\lg n)^k = O(n^4)$？

**答案：True（真）**

**證明：** 對任意常數 $k > 0$ 和任意 $\epsilon > 0$，都有 $(\lg n)^k = o(n^\epsilon)$。

**方法一：取對數比較**

$$\lg((\lg n)^k) = k \cdot \lg(\lg n)$$

$$\lg(n^4) = 4 \cdot \lg n$$

因為 $\lim_{n \to \infty} \frac{\lg(\lg n)}{\lg n} = 0$，所以 $k \cdot \lg(\lg n) \ll 4 \cdot \lg n$

**方法二：L'Hôpital 法則**

$$\lim_{n \to \infty} \frac{(\lg n)^k}{n^4} = \lim_{n \to \infty} \frac{k(\lg n)^{k-1} \cdot \frac{1}{n \ln 2}}{4n^3} = 0$$

因此 $(\lg n)^k = o(n^4)$，自然也有 $(\lg n)^k = O(n^4)$。∎
